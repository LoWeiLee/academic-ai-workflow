# 05_輸出/

**用途**：所有 AI 產出的唯一寫入區。無論是文獻分析、綜整報告、章節草稿、審稿診斷或編碼結果，AI 在任務執行過程中的新產出一律先寫入本夾，不直接寫入 00–04 各夾。

**讀寫紀律**：本夾為可寫入區，依專案建立子資料夾（如 `paper1/`、`paper2/`）分流存放，避免不同專案產出混雜。經你確認產出內容無誤後，可手動將定案版本搬移至 01–04 對應夾歸檔；本夾本身可視為工作暫存區，不做為最終定案位置。

**檔名規則**：`專案名稱_內容類型_v1.副檔名`（如 `paper1_文獻綜整_v1.md`）；同名檔案再次產出時遞增版號（v2、v3…），不覆蓋既有檔案，以保留每次產出的歷史紀錄。
