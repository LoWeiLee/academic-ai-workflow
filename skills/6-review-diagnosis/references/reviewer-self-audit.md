# 審稿者自檢（reviewer self-audit）

## 三鏡頭（五面向打分前的判斷層）

每篇同時用三鏡頭看，建議順序「先貢獻再方法」以避免方法隧道視野：

- **鏡頭 1 內部效度「證據支持主張嗎」**：中心主張與證據鏈是否成立——具體判準直接用 `reasoning-framework.md`（Toulmin 的 Warrant、IBE 替代解釋），本處不重複；獨有檢查點：抽掉任一證據論證會垮嗎（會→單點支撐，標記）。
- **鏡頭 2 外部效度「除了本研究還重要嗎」**：母體？樣本代表母體嗎？條件可複製？換情境/文化/時代仍成立？作者沒提的邊界條件？捷思：多數作者高估可推廣性；單一機構/單國樣本不得宣稱「普遍」。
- **鏡頭 3 貢獻「so what」**：本文之前知道什麼→之後知道什麼→delta 有意義（非只統計顯著）嗎→誰受益→開了什麼新問題。捷思：一句話講不出 delta，要嘛貢獻弱、要嘛沒講清楚，兩者都值得寫進意見。

## 八類審稿者偏誤（輸出前逐條自問「我有沒有犯」）

| 偏誤 | 描述 | 如何避免 |
|------|------|---------|
| 過度批判 | 放大小問題、忽略整體貢獻 | 先肯定優點再指問題，區分重大/次要 |
| 確認偏誤 | 只找支持自己既有觀點的證據 | 刻意找本文優點與自己觀點的反例 |
| 偏好投射 | 要求作者用「我的方法」 | 問「這方法能回答問題嗎」，非「我會怎麼做」 |
| 典範偏誤 | 用量化標準評質性（或反之） | 用與本文研究典範相符的標準 |
| 權威偏誤 | 因作者單位/資歷放鬆標準 | 只看稿件本身品質 |
| 新穎性偏誤 | 低估複製/漸進研究 | 承認複製對科學的價值 |
| 長度偏誤 | 長=好、短=草率 | 看內容密度非頁數 |
| 語言歧視 | 因非母語不完美而低估研究品質 | 區分「語言待潤」與「研究品質差」 |

## 校準五題（擬完審稿意見後自答）

1. 若照現狀發表，會誤導讀者嗎？（會→大修或退稿）
2. 作者能在一個修訂週期內合理處理我的關切嗎？（不能→退稿）
3. 我對這篇是不是比對自己的研究更嚴苛？（校準檢查）
4. 我有沒有指出至少一個真實優點？（平衡檢查）
5. 即使退稿，我的意見能幫作者改進嗎？（建設性檢查）

## 與五面向／讓步閾值的關係

三鏡頭是五面向打分前的判斷層；八偏誤＋校準五題是輸出前的自我把關。Mode C（自審）最易犯確認偏誤與過度寬鬆，校準第 3 題尤其重要。本自檢不改變「整體取最低面向為上限」與審查後對話的讓步閾值規則。論證強度判斷（Toulmin／因果／知識論狀態）見 `references/reasoning-framework.md`。
