# review-diagnosis｜評估清單參考（method-checklists）

> 載入時機：第一階段（五面向問題集）與第二階段（各章評估重點）。SKILL.md 守決策層與流程，本檔提供逐項評估的問題清單與檢查重點，於對應階段才載入。Mode A、B、C 共用；嚴格度層級依稿件類型套用（見 SKILL.md 嚴格度層級表）。

---

## 第一階段：五面向問題集

每一面向採「簡短摘錄 + 簡答」格式：先從原文摘錄關鍵句作為證據（含頁碼），再以一兩句話回答。最後給該面向的綜合評等。

**面向一：價值定位（對應緒論章）**

```
Q1.1 這篇文章的價值在哪裡（實務價值、理論價值）？
Q1.2 有錨定出研究缺口嗎？是實務缺口還是理論缺口？
Q1.3 有具體的貢獻聲明嗎？貢獻的性質為何（概念澄清/情境補充/理論修正/方法貢獻）？
Q1.4 研究問題是什麼？有說清楚嗎？問題與目的之間是否一致？
Q1.5 核心概念是否有操作型定義？（特別重要：易被忽略的根本性檢查）
```

**面向二：文獻運用（對應文獻回顧章）**

```
Q2.1 文獻是否有助於回答研究問題？還是只是教科書式堆砌？
Q2.2 文獻是否有助於建立研究架構（推導出假設或分析框架）？
Q2.3 文獻是否有助於導出研究方法的選用？
Q2.4 文獻有做到對話還是只是堆疊？（A 說、B 說、C 說 vs. 整理出爭論並表態）
Q2.5 文獻引用的時序一致性：結果章引用的文獻，是否在文獻章已預先建立？（檢查「事後找文獻來掛」現象）
```

**面向三：研究設計（對應研究方法章）**

```
Q3.1 研究設計是否有助於回答研究問題？（方法與問題的匹配度）
Q3.2 研究方法的選擇是否有理由？是否說明為何選此方法而非替代方案？
Q3.3 樣本（量化）/受訪者（質性）/資料來源（其他）是否具有代表性？選樣邏輯為何？
Q3.4 研究方法有重大瑕疵嗎？信效度（量化）或反身性（質性）是否被處理？
Q3.5 【質性個案】個案邏輯選擇（典型/關鍵/極端）是否清楚？外部推論策略為何？回顧性合理化偏誤是否被處理？
```

**面向四：研究結果（對應結果與分析章）**

```
Q4.1 結果的呈現是否清楚？是否系統且完整？
Q4.2 是否存在過度詮釋（相關被當因果、統計顯著被當理論重要）？
Q4.3 是否有特殊發現？（與理論預期或實務認知的差異之處）
Q4.4 不顯著或方向相反的結果是否被如實討論，還是被輕描淡寫？
Q4.5 詮釋深度：是停留在「一階確認」（訪談印證理論），還是進入「二階詮釋」（為什麼會印證、印證的條件是什麼）？
```

**面向五：結論與貢獻（對應結論建議章）**

```
Q5.1 研究結論是否來自於調查資料所導出？還是超出資料的推測？
Q5.2 是否存在過度詮釋？結論的推論強度與研究設計是否相符？（Mode C：以 paper-structure.md 設定的宣稱強度為對照）
Q5.3 結論是否完整回答了緒論的研究問題？（逐一對應）
Q5.4 是否做到與理論對話（支持/挑戰/修正了哪些主張）？
Q5.5 貢獻聲明與研究限制是否誠實交代？特別是研究者角色衝突、敘事偏誤、樣本侷限等關鍵限制是否被坦誠提及？
```

## 第二階段：各章節評估重點

依每章的評估重點清單逐一診斷，補充第一階段未覆蓋的細節。章節順序對應五面向，便於交叉檢視。

**若第一階段任一面向被評為「嚴重缺陷」**：本階段聚焦於確認該面向的根本問題成因——不是要找出所有缺陷，而是要理解「為什麼這篇文章在這個面向有根本性問題」。過度窮舉細節缺陷不僅無益，還會讓審查意見失焦。

### 第 1 章：緒論（對應面向一）

- 研究背景是否精煉聚焦，還是流水帳式羅列？
- 研究動機是否明確區分了「背景（現象存在）」與「動機（為何現在、為何此問題、既有研究為何無法回答）」？
- 研究缺口的指出：是實務缺口還是理論缺口？是否清楚指出既有研究的不足，且不過度誇大？
- 研究問題是否具體、可被研究設計回答、並具備學術迫切性？
- **核心概念的操作型定義**：被探討的對象（如「成功」、「韌性」、「治理能力」）是否有明確的操作型定義？多重指標時是否說明本研究採用哪一（些）？
- 研究目的之間的內部一致性（多目的論文常出現相互牽制）

### 第 2 章：文獻回顧（對應面向二）

- 文獻選擇是否以學術相關性為標準？是否涵蓋近五至七年的重要發展？
- 文獻是否圍繞研究問題，還是過度擴張為教科書式堆砌？
- 是否整理出理論間的爭論，而非逐篇描述（A 說、B 說）？
- 是否自然推導出研究框架或假設？
- **構念邊界區分**：跨理論借用時，是否明確說明各構念的理論歸屬與邊界？（如 TOE/TAM/IDT/UTAUT 構念混雜的問題）
- **特殊場域的理論調適**：若研究場域為政府、非營利、特定產業等特殊脈絡，是否討論理論在該場域的適用性與調適？
- **文獻嵌入時序一致性**：結果或分析章引用的文獻，是否在文獻章已預先建立？避免「事後找文獻來掛」

### 第 3 章：研究設計與方法（對應面向三）

> **質性研究的核心檢查整合於此**：質性反身性核心化、不對少數樣本統計平均、個案邏輯與外部推論、主導性受訪者敘事偏誤四項為本章評估重點，不另列獨立清單；審查質性稿件時逐項落實於下方【質性研究】與【質性—個案研究次分類】。

**評估重點（依方法論類型調整）**：

**【量化研究】**
- 方法選擇的理由是否被明確說明（如為何選 PLS-SEM 而非 CB-SEM）？
- 研究工具（量表）是否有足夠的信效度依據（建構效度、收斂效度、區別效度）？
- 抽樣程序是否被清楚描述且具代表性？
- 統計分析策略是否符合資料性質（如連續 vs. 類別變數）？

**【質性研究】**——核心檢查項
- 質性方法選擇的理由是否被說明（為何訪談而非問卷，此方法如何捕捉深層機制）？
- 資料飽和度如何判斷（理論飽和 vs. 主題重複）？
- **研究者反身性（reflexivity）**【核心檢查項】：研究者與研究對象的關係是否被坦誠交代？研究者的位置（同事、上下屬、政策參與者）如何影響訪談取徑、編碼判斷與詮釋方向？是否有具體做法降低偏誤（訪談大綱第三方檢視、錄音交叉聽取等）？
- 分析程序（編碼邏輯、主題歸納策略）是否透明？成員檢核（member checking）或同儕稽核是否被執行？
- **主導性受訪者敘事偏誤**：多位受訪者中，是否有單一受訪者話語比例過高、主導敘事？這是常見但易被忽略的偏誤
- ⚠️ **方法論提醒**：若樣本為少數質性訪談對象，不應對回應進行統計平均或百分比計算

**【質性研究 — 個案研究次分類】**
若本研究為單一個案或少數個案研究，補充以下檢查：
- 個案邏輯選擇：典型個案（typical）/ 關鍵個案（critical）/ 極端個案（extreme）/ 偏離個案（deviant）？選擇邏輯是否清楚？
- 外部推論策略：是統計性推論、分析性推論、還是理論性推論？對應的推論範圍為何？
- **回顧性合理化偏誤**：若選擇成功個案研究，受訪者容易事後將成功歸因於某些因素，研究設計是否有機制處理此偏誤？
- **倖存者偏誤**：未研究失敗或部分成功個案作對照，能否區分「個案特殊條件」與「普遍規律」？
- 三角檢證的執行情況（不只是宣稱有，而是具體做了什麼資料來源的交叉驗證）

**【混合方法】**
- 混合設計的整合邏輯是否清楚？量化與質性部分在何處整合，整合目的為何（三角驗證 / 深化解釋 / 擴展發現）？
- 兩個部分各自的信效度標準是否都被討論？

**【共用項目】**
- 研究設計是否與研究問題的性質（探索性 / 解釋性 / 描述性）相符？
- 是否提及研究倫理（IRB 或知情同意）？

### 第 4 章：研究結果與討論（對應面向四）

- 結果呈現是否系統且完整（描述統計 + 推論統計 / 質性主題分布）？
- 是否同時報告統計顯著性與效果量（β、Cohen's d、R²）？
- 是否存在過度詮釋：相關關係被描述為因果關係？統計顯著被誤解為理論重要？
- 不顯著或方向相反的結果是否被如實討論？
- 結果是否明確與文獻回顧中建立的理論框架對話（支持、挑戰、或修正了哪些主張）？
- **一階確認 vs 二階詮釋深度**：分析是停留在「資料印證理論」（一階），還是進入「為什麼會印證、印證的條件是什麼、印證之外還有什麼」（二階）？
- **「非屬框架」發現的處理**：若有超出原框架的因素出現，是被當成補充處理，還是被詮釋為「原框架本身需要修正」的證據？前者削弱理論整合性，後者反而是潛在的理論貢獻

### 第 5 章：結論與建議（對應面向五）

- 結論是否完整回應緒論的研究問題？（逐一對應）
- 理論貢獻是否被清楚且具體地陳述（性質：概念澄清 / 情境補充 / 理論修正 / 方法貢獻 + 邊界條件）？
- 政策建議是否直接推導自研究發現，還是「順手附加」？
- **政策建議的場域差異化**：建議是否針對不同類型機關（中央/地方、大/小規模、有/無基礎條件）提供差異化策略，還是一般性泛論？
- **研究限制的誠實度**：質性研究中的研究者角色衝突、敘事偏誤、回顧性合理化偏誤等關鍵限制是否被坦誠提及，而非以「樣本有限，未來可擴大」類的敷衍式描述帶過？
- **（Mode C）宣稱強度一致性**：結論的推論強度是否與 paper-structure.md 設定的貢獻性質（探索性框架 / 理論建構 / 假說驗證）一致，未在收尾處悄悄升級。

### 形式品質檢查（獨立區塊，與五面向並行）

不影響學術品質但影響可接受度的形式問題，於審查意見中以獨立區塊呈現：

- **PDF 字元異常**：是否有字型嵌入問題、字元編碼問題、OCR 錯誤？（建議在不同 PDF 閱讀器中驗證）
- **錯字與全文校對**：是否有明顯的中文輸入錯誤、英文拼字錯誤、標點誤用？
- **章節結構順序合理性**：章節安排是否符合「方法在資料之前」的一般邏輯？
- **參考文獻體例一致性**：APA / 學校規範是否一致？中英文文獻處理是否一致？
- **圖表編號與內文引用一致性**：圖表編號連續、圖表內容與內文敘述一致
- **（Mode C）writing-standards.md 禁止清單比對**：空洞開場與過渡、貢獻宣稱虛化、因果語氣濫用、AI 寫作痕跡、不當粗體/副標/emoji 等是否出現
